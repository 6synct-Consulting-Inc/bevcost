===================================
Welcome to bevcost's documentation!
===================================

**bevcost** (/BEV cost/) is a Python package that simplifies the total cost of ownership (TCO)
analysis of BEV fleets in mining. It is for engineers, operations analysts and consultants to 
support decision-making and techno-economic analyses.

Developed as part of the `EVMO Project <https://zero.nexus/evmo/>`__.

.. note::

	This project is under active development.
